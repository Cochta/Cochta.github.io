This is the documentation of my network code for my online multiplayer turn-based game.

By reading this, you will understand how to use my protocol.

Creating a new Packet

To send data between server and clients i use an enum called PacketType.

The different types allow different data types to be sent and to be handled by the logic.

PacketType:

  • Connect
  • StartGame
  • HasPlayed
  • GameFinished
  • Surrender
  • QuitLobby
  • Invalid

To create a new packet, some things must be done first.

First, enter the new packet into the enum, for exemple: NewPacket.

Then, create a new struct with Packet as final heritance and put all desired data in it.

struct NewPacket final : Packet
{
NewPacket() : Packet(PacketType::NewPacket) {}
int data = 0;
};

After, override the << and >> operators.

sf::Packet& operator<<(sf::Packet& packet, const NewPacket& content) {
return packet << content.data;
}
sf::Packet& operator>>(sf::Packet& packet, NewPacket& content) {
packet >> content.data;
return packet;
}

Then, add the NewPacket in the 3 switch cases found in Packet.cpp

In the FromType:

case PacketType::NewPacket:
return new NewPacket();

In the sf::Packet& operator<<

case PacketType::NewPacket: {
const auto& newPacket = dynamic_cast(packetType);
packet << newPacket;
break;
}

In the sf::Packet& operator>>

case PacketType::NewPacket: {
auto* newPacket = dynamic_cast(&packetType);
packet >> *newPacket;
break;
}

Network logic

Server

To execute code by the server when a packet is received, just put your logic in the following lambda.

server.ListenToClientPackets([](sf::TcpSocket* socket,Packet* packet) {
if (packet->type == PacketType::NewPacket) {
// your code logic
}
});

Client

To execute code by the client their are some additionnal steps.

First, at the start of the main loop, loop throught all the received packets, execute your code and pop them.

while (Packet* packet = _client.PopPacket()) {
if (packet->type == PacketType::NewPacket) {
// your code logic
}
}
